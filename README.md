# GeometricAlgebra

## Installation

### Locally

```
/build.wls; ./install.wls
```

### Or with released paclet link
```
PacletInstall["https://github.com/sw1sh/GeometricAlgebra/releases/download/0.8/GeometricAlgebra-0.8.paclet"]
```
